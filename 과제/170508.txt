grep:파일에서 특정 문자열 찾기
 - grep [옵션] 패턴 [파일명]
  	^:행의 시작
	$:행의 끝
	.:한글자
	[]:괄호안의 문자 중 하나
	[^]:괄호안의 문자가 아닌 한 문자
	?:한글자(없어도 됨)
	*:없거나 여러 자

egrep:grep+특수문자 추가
	+:앞의 글자가 하나 이상 나옴
	x|y:x나y중 하나
	(|):그룹 만들기

fgrep:문자열로만 검색(특수문자,정규식사용불가)

find : 파일검색
 - find [경로] 검색조건 [동작]
	-name:이름으로 파일 찾기
	-type:파일종류로 검색(d:디렉토리/b:블록장치파일/f:일반파일/c:문자장치파일/l:심볼릭링크/s:소켓)
	-atime:파일접근시간으로 검색
	-mtime:파일수정시간으로 검색
	-newer:특정파일의 수정시간으로 검색
	-perm:파일 접근권한으로 검색
	-user:파일소유자로 검색

whereis : 특정명령의 절대경로 위치를 출력

which : 명령어 파일의 위치를 찾아서 그 경로나 앨리어스를 출력
