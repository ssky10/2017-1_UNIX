vi 편집기
- UNIX,LINUX에서 사용하는 편집기(=윈도우의 메모장)

명령모드/입력모드
명령모드 - : / ?
입력모드 - i a o O
입력모드 탈출 - esc

기본사용명령
저장 - :w [파일명]
파일생성 후 입력 - vi 파일명
종료 - :q[!]
저장후 종료 - :wq [파일명]

커서 이동(명령모드에서 실행)
방향으로 이동 - h(좌)j(하)l(상)k(우) or 방향키
줄의 시작으로 이동 - ^ 또는 0


데이터 삭제(잘라내기)
x -> delete
X -> backspace
dw -> 단어단위 삭제
D -> 현재 커서부터 맨끝까지 삭제
dd -> 현재 줄 완전삭제
:<번호>d -> 번호에 해당하는 줄 삭제
u -> 되돌리기

복사/붙여넣기
줄 복사 - <범위>yy
해당 줄 복사 - :<줄번호>y
붙여넣기 - p
커서 다음 줄에 붙여넣기 - :pu
다음 줄에 붙여넣기 - :<줄번호>pu

문자열 탐색
정방향 탐색 - /찾고자 하는 문자열
역방향 탐색 - ?찾고자 하는 문자열
다음 탐색 - n
다음 역탐색 - N

문자열 치환
:s/문자열1/문자열2 -> 커서가 위치한 행의 첫번째 문자열1을 문자열2로 바꾼다.
:%s/문자열1/문자열2/g -> 파일전체에서 모든 문자열1을 문자열2로 바꾼다.
:<범위>s/문자열1/문자열2 -> 범위안의 모든 행에 대하여 각 행의 첫 번째 문자열1을 찾아 문자열2로 바꾼다
:<범위>s/문자열1/문자열2/g -> 범위안의 모든 행에서 문자열1을 문자열2로 바꾼다.
:<범위>s/문자열1/문자열2/gc -> 범위안의 모든 행에 대하여 각 문자열1을 문자열2로 바꿀때 수정할지 여부를 묻는다.
