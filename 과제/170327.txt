*cat [옵션] [파일명]
 - 파일명이 없을 경우 : 사용자가 입력한 값을 그대로 출력해준다.
   # ctrl+D(종료)를 이용해서 빠져나감
 - 파일명이 있는 경우 : 해당파일의 내용을 출력한다.
 - cat > 파일명 : 파일명에 해당하는 파일을 작성/생성한다.
   # > : 출력리다이렉션(좌측명령의 출력값이 우측으로 들어간다.)
 - 행번호 출력 : -b/-n

*more [옵션] 파일명 : 파일의 내용을 출력한다.(길이가 길 경우 나누어 출력)
 - 다음페이지 : space
 - 이전페이지 : b
 - 종료 : q
 - 문자열 찾기 : /문자열
 - 문자열 계속 찾기 : n

*head [옵션] 파일명 : 파일의 내용을 앞에서 부터 특정 줄(기본10줄)만큼 출력
 - 지정 줄 수 출력 : -n 줄수

*tail [옵션] 파일명 : 파일의 내용을 앞에서 부터 특정 줄(기본10줄)만큼 출력
 - 지정 줄 수 출력 : -n 줄수
 - 파일 감시 : -f

*cp [옵션] 파일(디렉토리)명1 파일(디렉토리)명2
 - 파일명1에 해당하는 파일을 복사하여 파일2라는 이름으로 붙여넣는다.
   (파일2가 존재할 경우 덮어쓰기)
 - 덮어쓰기여부 확인 : -i
 - 디렉토리를 복사할 때 : -r

*rm [옵션] 파일(디렉토리)명
 - 삭제 여부 : -i
 - 디렉토리를 삭제할 때 : -r
 - 무조건 삭제 : -f

*ln [옵션] 원래이름 새이름 : 링크생성
 - 하드링크(기본값) : 파일의 메모리 주소를 이용해 링크가 걸린다.(디렉토리 불가)
 - 심볼릭 링크(-s) : 파일의이름을 이용해 링크가 걸린다.(원본이 삭제될 경우 접근불가./디렉토리 가능)

*touch [acm옵션] [rt옵션] 파일
 - 파일의 접근 시간을 변경해주는 명령 / 더미파일을 생성해주는 명령
 - 접근 시간 변경 : -a
 - 지정된 파일이 없는 경우 : -c
 - 수정 시각만 변경 : -m
 - -r ref_file : 시각을 파일에서 읽음
 - -t [[[CC]YY]MMDDhhmm[.ss]] : 시각을 직접 입력
