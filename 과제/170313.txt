유닉스란?
컴퓨터 운영체제의 일종으로 주로 서버구축에 사용

유닉스의 특성
-대화형 시스템
-다중 사용자 시스템
-다중 작업용 시스템
-높은 이식성과 확장성

유닉스 접속하기
id@class.gnu.ac.kr | 203.255.3.232 | class.gnu.ac.kr

명령 형식
명령 [옵션] [인자]
 - 명령은 '-'기호로 시작

프롬프트 기호
-셸의 종류에 따라 상이
  - $ : 본셸/콘셸/배시셸/대시셸
  - % : C셸

기본 명령어
- date : 현재 설정시간을 나타냄(옵션에 따라서 설정도 가능)
- whoami : 현재 접속 사용자의 이름을 보여줌
- man [옵션(페이지 번호)] 명령어: 명령어에 대한 매뉴얼을 보여준다. 
- who : 현재 접속중인 사용자를 보여줌.
- clear : 화면을 정리
- ls : 현재 디렉토리의 파일을 보여줌
  - 옵션 : a:숨김파일 보기
           l:파일세부정보 보기
  * ls -al(본셸) == ll(배쉬셸)
- logout : 로그아웃
- exit : 종료
- passwd : 패스워드 변경
- stty -a : 모든 특수문자(의 설정)을 출력
- stty 변경 옵션명 메타문자 : 변경옵션에 해당하는 메타문자을 변경
