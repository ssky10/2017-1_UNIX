디렉토리란?
- 윈도우의 폴더와 같은 역할 수행, 하나의 파일로 인식

디렉토리 계층구조
/(루트) -> 디렉토리 -> 파일

절대경로와 상대경로
- 루트로 부터 경로 위치의 표시(현재 내 위치 영향X) : 절대경로
  * 경로가 명확하여 중요한 작업시 권장
- 내가 현재 있는곳으로 부터 위치의 표시 : 상대경로
  * 현재 디렉토리 : .
  * 상위 디렉토리 : ..
  * 홈 디렉토리 : ~
  * 하위 디렉토리 : 이름

파일과 디렉토리 명 주의사항
- /를 제외한 모든 문자 사용가능
- 알파벳,숫자,하이픈(-),언더바(_),점(.)만 사용
- 공백은 포함되지 않음
- 알파벳은 대소문자 구분해 사용
- 이름의 시작이 .으로 시작하면 숨김파일로 간주

- pwd : 현재 내위치 보여줌

- cd 디렉토리명 : 해당 디렉토리로 이동

- ls [옵션] [디렉토리명]
  1.디렉토리명 생략시 현재 디렉토리 목록
  2.디렉토리명 지정시 지정 디렉토리 확인
  3.-a/-F 파일정보 확인(-a:숨김파일포함/-F:기본파일(디폴트값))
  4.-l:상세 정보 보기
  5.-ld:디렉토리 자체 정보 확인

- mkdir [옵션] 디렉토리명 : 디렉토리 생성

- rmdir [옵션] 디렉토리명 : 디렉토리 제거

- mv 디렉토리1 디렉토리2
  1. 디렉토리2가 존재할 경우 : 디렉토리 2안으로 이동
  2. 디렉토리2가 없을 경우 : 디렉토리1의 이름이 디렉토리2로 변경

- mv 파일 디렉토리 : 디렉토리 안으로 파일 이동

- mv 파일1 파일2 : 파일1의 이름을 파일2로 변경
