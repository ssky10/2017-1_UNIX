read : 표준입력으로부터 값을 읽어 변수에 할당
 read 변수1 ....

expr : 수식을 계산하여 결과를 화면에 출력
 `expr 수식`
 - 수식의 모든 요소는 공백으로 분리
 - 쉘 메타문자(*,|,>,<,(,),&)앞에 \를 붙여서 메타문자 기능 상실시켜야함

제어문
 for 변수 in 단어1 단어2 ... 단어n
 do
   명령 ...
 done
 - for((초기화,조건,변경식))으로 대체 가능(bash)
 - for 변수 in {시작..끝}으로 대체 가능(bash)
 while 조건식
 do
  명령 ...
 done
